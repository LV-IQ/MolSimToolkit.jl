# MolSimToolkit.jl

[MolSimToolkit.jl](https://github.com/m3g/MolSimToolkit.jl) is in very early stages of development.

## Installation

```julia
import Pkg; Pkg.add("MolSimToolkit")
```

## Using

```julia
using MolSimToolkit
```

## Recommended

Also install `PDBTools`:

```julia
import Pkg; Pkg.add("PDBTools")
import PDBTools
```
